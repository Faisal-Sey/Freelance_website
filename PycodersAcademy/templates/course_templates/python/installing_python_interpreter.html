Python Coding

Python Programming is one of the popular languages among the programming languages. It is an interpreted language meaning that, it does not require any compilation process when running the program. The main process in running python codes is just hitting the run button or running it from the command line. This feature of the python programming language is performed by its interpreter.

Selecting your preferred Python Interpreter

The python interpreter could be downloaded from the official website of the python programming language organization. There are two main stable releases of the python interpreter. They are:

  • The Python 2.XX
  • The Python 3.XX

Python 2.XX is the first release version of the Python interpreter and could be found pre-installed in some operating systems like the MAC OS (Operating System). The Python 3.XX is the latest and most stable release from the Python Programming language organization and has a huge community backing. The XX meaning the version of the Python 3 released. The latest version release of the Python 3.XX is Python 3.10 and for Python 2.XX the latest version is Python 2.7.18.

A list of all Python releases can be viewed here

Downloading the Python Interpreter

To download the Python interpreter on your PC, there are some steps to follow to get the correct installer.

STEP 1:

Check the Operating System you are installing on. Python is cross-platform hence could be used on all Operating System but their installations differ from each other.

STEP 2:

Check the processor type of the Operating System. The are two types of processors depending on the Operating System architecture. They are 64 bits OS and 32 bits OS. Operating System on 64 bits has better experience than the 32 bits. Softwares that runs on 32 bits OS can also run on 64 bits OS but softwares that are made specifically for 64 bits can't run on 32 bits OS.

STEP 3:

Check the type of Python interpreter that is favourable for the work you are using it for. Some versions may luck features which you may need in some specific area of application.

After treating the above steps, we can now download the version of our choice which is compatible for our OS. We can now head to here and download the Python Interpreter version.

Installation of the Downloaded Python Version

After downloading for the specific OS, we look at how to install it on the OS. On this page, Installation on windows will be treated and links will be provided to install for those on the other OS.

Installing Python on Windows

About 45.8% of Python Programmers use windows as written here, hence it very important to look at it installation on windows. There are a lot of ways to install Python on windows, our main focus will be on installing from an executable file downloaded from the python official website. Installing from an executable file is very simple.

  • Double tap on the file to start installation process.


  • In the image above, there are two installation directories you can install python. The first being the default path and the second being the customize path. The default path is chose by the python installer and the customize path allows the use to install the software on the user's preferred location.
    At the bottom path of the installer window is two checkboxes. The first checkbox when checked, installs the Python software for all users on the OS. The second checkbox when selected adds Python to the environment variables/path which makes it recognizable for other in built apps like the command prompt

  • Click on next after setting the preferred configurations to install the software.

  • After installation, 2 softwares and 2 docs files is added to the OS. the two softwares are: IDLE, Python
    And the two doc files are: Python manuals and Python module docs

  • IDLE - is an in built editor to write and run python codes.
    The Python software - is the interpreter that runs the codes.
    Python manuals - is the official document for the version.
    Python module docs - is the document for the in built modules.

Okay great! So Python is installed successfully. To check that it is has been added to the system path, open command prompt and type python or python3.
