In our previous lesson, we look at downloading and installing the python important softwares. We looked at the IDLE, which we said it an editor to write python codes. In this lesson, we look at an editor which can serve the same purpose as the IDLE and even better.

What is an Editor

Editors, normally known as Integrated Development Environment (IDE), is an environment that provides you with a space to write your codes. This implies that, to create softwares or programs using python programming language, a software in which our codes will be written in is needed.

Types of Editors

There are two main types of editors for writing Python Programming codes. They are the Writing only Editors and the Writing and Running Editors

Writing only Editors

This is the first type of editor and existence long before the first of the second type of editors was published. The Writing only Editors are only used for writing of the codes. The process of display of the output from the code becomes tedious. The stages involved in running codes from Writing only Editors are as follows:

  • The Code is written in the Writing only Editor.
  • Third party software such as the windows command prompt is then used to run the written code.
  • An output is displayed in the software after a successful run.

Example of this type of Editor is Notepad++

Writing and Run Editors

This is the modern type of editors. This type of editor by default detects the python interpreter then passes the code written in the editor to it. These types of editors makes Python Programming very easy and also very efficient since realtime result is received as the coding is in process. The stages involved in running codes from Writing and Run Editors are as follows:

  • The Code is written in the editing section of the Editor.
  • A button present in the editor is clicked to run the code.
  • An output is displayed on the console of the editor.

Example of this type of Editor is Pycharm.

Downloading the Pycharm Editor

The Pycharm Editor is used for both writing and running of codes. It heavily supports the Python Programming Language alongside other Programming languages and also markup languages including HTML. It has a lot of features which makes it easy to use.

Features of the Pycharm Editor

For full features of the Pycharm Editor, visit here Below are some popular features of the Pycharm Editor.

  • Support for other Programming Languages.
  • Github tool to push and update your code, among others.
  • Support for Django and other Web Framework.
